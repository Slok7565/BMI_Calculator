# The Most Interesting BMI Calculator

## Overview

The Most Interesting BMI Calculator is not your ordinary health toolˇŞit's an interactive, personality-driven Python application that calculates your Body Mass Index while providing a unique and entertaining experience. Created by Slok, this calculator combines practical health metrics with hidden easter eggs, character interactions, and humorous responses.

## Features

- **Interactive Typewriter Effect**: All text appears with a classic terminal typewriter animation
- **Character Easter Eggs**: Discover hidden interactions with characters from Undertale and Deltarune
- **Weekly Tracking**: Calculate and track your BMI across a full week (7 days)
- **Personalized Responses**: The calculator adapts its responses based on your name and inputs
- **Error Handling with Personality**: Even incorrect inputs generate entertaining responses
- **Special Effects**: Unique visual and text effects for certain interactions

## How to Use

1. Run the Python script in your terminal
2. Enter your name when prompted (try some creative names for special interactions!)
3. Follow the instructions to input your weight (in kg) and height (in m) for each day
4. Review your weekly BMI results at the end

## Character Easter Eggs

The calculator includes special interactions with characters from popular games:

- **Undertale Characters**: Toriel, Sans, Papyrus, Undyne, Alphys, Mettaton, Asgore, Flowey, and more
- **Deltarune Characters**: Kris, Susie, Ralsei, Noelle, Berdly, Queen, Spamton, Jevil, and others
- **Special Mentions**: Gaster, Chara, Frisk, and other hidden characters

Each character responds in their signature style, with dialogue prefixed by asterisks (*) just like in the games.

## Technical Requirements

- Python 3.x
- Windows OS (for msvcrt module functionality)
- No additional dependencies required

## Code Structure

The application is built with several key functions:

- `type_effect()`: Creates the typewriter animation for text output
- `input_with_effect()`: Handles user input with visual effects
- Easter egg detection and response system
- BMI calculation and weekly tracking logic

## Special Notes

- Try entering names like "Gaster" for a particularly interesting experience
- Some names trigger special responses and even rename the user
- Multiple incorrect inputs lead to increasingly frustrated responses from the calculator

## Disclaimer

This BMI calculator is designed for entertainment purposes alongside health metrics. For serious health concerns, please consult medical professionals. The BMI index is a general guideline and may not be appropriate for all body types.
